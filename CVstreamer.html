CVstreamer

Language

Python 3.9

Libraries

OpenCv, Dlib, Pygame, Numpy

Tools

VScode, Adobe Illustrator

The idea was to integrate my knowledge of facial point detection in OpenCv and the ability to create a 2d environment in Pygame. I created this motion capture program with the help of powerful libraries like OpenCv and Pygame. The code captures the users facial points using the camera and determines facial landmarks using a .dat file , then calculates the distance of seperation between important points to determine if the user is opening/closing his eyes/mouth. These calculations are passed through the Pygame code which converts numbers to expression using simple conditional logic.

Applications

Some of the real time application are:

  • Virtual Streamers to stream 2d characters(Youtube/Twitch).
  • Filters for Social Media (Snapchat/Instagram/Tiktok).
  • Can be used in messaging apps to promote fun interactions between users.
  • To create 2d movies without actually needing to animate everthing.
